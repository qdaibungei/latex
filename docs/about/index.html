文藝部LaTeX研究会とは

九州大学文藝部の下部組織。組版を中心とした文藝部の技術的側面を研究する目的のもとに誕生した。現在、研究会員は約一名おり、主にLaTeXの研究に取り組んでいる。

会員は常時募集中である。
