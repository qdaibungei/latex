### 文藝部LaTeX研究会とは
九州大学文藝部の下部組織。組版を中心とした文藝部の技術的側面を研究する目的のもとに誕生した。LaTeXなどの研究を行っている人が一名ほど所属していたりする。

このサイトではその研究資料などを公開する。

### CSS
